# AI Voices Tracker

AI Voices Tracker is a Chrome extension for tracking AI leaders, founders, researchers, investors, engineers, and other notable voices across the AI ecosystem.

## Purpose

Instead of bookmarking profiles or maintaining spreadsheets, AI Voices Tracker provides a lightweight way to build and organize a personal database of influential people discovered while browsing.

## Features

- Track AI people with one click
- Store profiles locally
- Organize notable voices
- Lightweight and privacy-focused
- Chrome Extension (Manifest V3)

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extensions API
- Manifest V3
